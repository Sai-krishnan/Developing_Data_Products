Executive Summary

The requirement of this project is to analyze the MTCARS dataset and answer the following questions:-

  1. Is an automatic or manual transmission better for MPG

  2. Quantify the MPG difference between automatic and manual transmissions

Each row in the MTCARS dataset lists the following parameters pertaining to different car models:-

[, 1] mpg Miles/(US) gallon
[, 2] cyl Number of cylinders
[, 3] disp Displacement (cu.in.)
[, 4] hp Gross horsepower
[, 5] drat Rear axle ratio
[, 6] wt Weight (lb/1000)
[, 7] qsec 1/4 mile time
[, 8] vs V/S
[, 9] am Transmission (0 = automatic, 1 = manual)
[,10] gear Number of forward gears
[,11] carb Number of carburetors

We explore the relationship of MPG versus cylinders, horsepower and weight the variables with both manual and automatic transmission by building multi regression linear models.

Manual transmission cars have a better MPG than auto transmission. The quntitative difference between the models is obtained by subtracting the respective eqautions. This is documented in the results section.

Let us try to compare the MPG differences due to type in transmission corresponding to each treatment statistically. We conduct a t test to compare the MPG corresponding to automatic and manual transmissions respectively. The null hypothesis is that there is no difference in the mean MPG of manual transmission and automatic transmission cars.

t.test(autoTrans$mpg - manualTrans$mpg)

Based on the t test, we can reject the null hypothesis that the mean MPG for automatic and manual transmission cars are same. We can conclude that the mean MPG for cars with manual transmission is better than the MPG of cars with automatic transmission.

Let us build models of MPG versus parameters like number of cylinders (cyl), horsepower (hp), weight (wt).

# Linear models for cars with automatic transmission
fitautompg <- lm(autoTrans$mpg~factor(autoTrans$cyl))
fit1autompg <- update(fitautompg, autoTrans$mpg~factor(autoTrans$cyl) + autoTrans$hp)
fit2autompg <- update(fitautompg, autoTrans$mpg~factor(autoTrans$cyl) + autoTrans$hp + autoTrans$wt)

# Linear models for cars with manual transmission
fitmanualmpg <- lm(manualTrans$mpg~factor(manualTrans$cyl) + manualTrans$hp)
fit1manualmpg <- update(fitmanualmpg,manualTrans$mpg~factor(manualTrans$cyl) + manualTrans$hp)
fit2manualmpg <- update(fitmanualmpg,manualTrans$mpg~factor(manualTrans$cyl) + manualTrans$hp + manualTrans$wt)

The coefficients of the following independent variables (from above equation) can be interpreted as follows:-

6CYL : Reduction in MPG as compared to a 4 cylinder engine
8CYL : Reduction in MPG as compared to a 4 cylinder engine
HP : Reduction in MPG due to unit increase in horsepower
WT : Reduction in MPG due to 1 tonne increase in weight

Results

Based on the analysis described above, cars with manual transmission result in better MPG than auto transmission. The models show that increase in weight, cylinders and horsepower lead to reduction in MPG. The difference between the MPG models for automatic and manual transmissions can be expressed as:-

MPG(MT) - MPG(AT) = 13.2422112 + 0.6376627 x6CYL + 4.7249567 x8CYL + 4.018012310^{-4}xHP -4.7700495xWT
